# Kashish Khubchandani Portfolio

## Run without assistant
Open `index.html` directly in browser.

## Run with AI assistant
1. Open this folder in VS Code.
2. Run: `npm install`
3. Rename `.env.example` to `.env`
4. Add your Groq key: `GROQ_API_KEY=your_key_here`
5. Run: `npm start`
6. Open: `http://localhost:3000`

## Files
- index.html = Home
- about.html = About + photo + education
- experience.html = Internship deep dive
- work.html = Projects with click detail modal
- skills.html = Skills and moving tools
- contact.html = Contact form + resume downloads
